# Конвертатор

## Входная информация

1) Расположение папки с файлами для конвертации (исходная папка), к примеру "J:\\Info"
2) Расположение папки для хранение данных, к примеру "E:\\Output"

## Цель

* Создание поиска по тексту

## Требования

* Необходимо сохранение папок в папке для хранения как в исходной папке.
* Если в исходной папке есть файл который можно сконвертировать, то необходимо сконвертировать и положить в папку для хранения данных.
* В папке для хранения данных не удаляются папки.
* Запуск программы может быть несколько раз, поэтому программа должна понимать - какие файлы ещё не сконвертированы.
* Файлов конвертации много поэтому конвертация должна быть параллельно производиться.
* Сконвертированные файлы - тесктовый документ в формате UTF8 с расширением ".txt"
* Язык программирование Golang
